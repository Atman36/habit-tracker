# **App Name**: Трекер Привычек

## Core Features:

- Добавление привычек: Интуитивно понятный интерфейс для легкого добавления новых привычек.
- Отметка выполнения: Простой способ отмечать выполнение привычек каждый день.
- Визуализация прогресса: Визуализация прогресса выполнения привычек с помощью графиков.
- Установка целей: Установка целей для каждой привычки и отслеживание их достижения.
- Уведомления: Получение уведомлений для поддержания регулярности выполнения привычек.
- Персональные советы: Получайте персонализированные советы по улучшению выполнения привычек. Этот инструмент анализирует данные о ваших успехах и трудностях и даёт подходящие подсказки.

## Style Guidelines:

- Основной цвет: Мягкий синий (#64B5F6) для спокойствия и концентрации.
- Цвет фона: Светло-серый (#ECEFF1) для снижения нагрузки на глаза.
- Акцент: Светло-зеленый (#A5D6A7) для индикации успеха и позитивных действий.
- Использовать удобочитаемые шрифты без засечек для улучшения восприятия информации.
- Применять минималистичные иконки для каждой привычки для улучшения навигации.
- Создать простой и понятный макет, ориентированный на быстрое добавление и отметку привычек.
- Добавить небольшие анимации при отметке выполнения привычки для поощрения пользователя.
